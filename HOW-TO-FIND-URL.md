# 如何找到你的网站链接

## 方法1：在 Vercel Dashboard 找

1. 访问 https://vercel.com/overseas-airdrop
2. 你会看到你部署的项目列表
3. 点击项目名称（可能叫 airdrop-hunter 或类似的）
4. 在项目页面顶部，会显示网址，类似：
   - `https://airdrop-hunter-xxx.vercel.app`
   - 或 `https://xxx-overseas-airdrop.vercel.app`

## 方法2：查看部署记录

1. 在 Vercel Dashboard，点击 "Deployments"
2. 找到最新的部署（状态显示 "Ready"）
3. 点击进入，顶部会显示 "Visit" 按钮
4. 点击 "Visit" 就能访问你的网站
